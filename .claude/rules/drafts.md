# Draft Writing Rules

When writing message drafts:

- Consult `context/voice/` first to match the user's tone and style
- Keep drafts concise and natural-sounding
- Don't sound like AI — write like the user writes
- Link text should be descriptive (e.g., "[the PR](url)", not "[link](url)")
- Always use full URLs — never bare references like "PR #123"
